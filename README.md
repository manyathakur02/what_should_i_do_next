# what_should_i_do_next
the objective is to create a web page or app where users get to know the best next step to achieve their goals

What it does:

Input screen — takes your goal, current level, time available, and any preferences
AI generation — calls Claude with web search enabled to build a real, researched roadmap
Results screen — a visual step-by-step roadmap with:

Numbered timeline with expandable cards
Duration estimates for each step
Topics covered per step
Real YouTube/website links found via live search
Pro tips for each step
Total plan duration
